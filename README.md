# OpenFlow 数据提取插件 (OpenFlow Data Extractor)

OpenFlow 是一款基于 Chrome Manifest V3 标准开发的浏览器扩展程序，专门用于自动化从特定的创意工单及资产管理页面中提取所需的各种维度数据信息，并一键生成规范的 CSV 或 JSON 文件。

## ✨ 核心特性

- **一键智能提取**：点击插件图标，自动注入脚本抓取当前页面的项目游戏名称、素材类型、尺寸要求明细、原数量与延展套数等核心数据。
- **全量信息捕获**：不仅抓取规格详情，还能在页面全文中自动捕获“键值对”信息（如：制作人、集团名称、投放媒体、预期完成时间、优先级等），最大限度补全数据。
- **动态列匹配CSV导出**：摆脱杂乱无章的 Excel 依赖！完全基于原生 Blob 生成干净的 UTF-8 编码 CSV 文件。
  - **平面模板**：自动套入《平面》定制化的特定表头和填充固定值；
  - **视频模板**：针对《视频》自动替换为专属字段，并套用自定义产出公式（如：自动推算 `视频总产出`）。
- **极客排序JSON导出**：JSON数据严格遵循核心必选字段前置策略，剩余补充元信息按字母序排列，数组信息托底保证最佳可读性。
- **免流转一键下载**：生成文件会自动采用当前提取到的“项目游戏名称”重命名下载（如 `小火车呜呜呜.csv`），拒绝文件名乱码和后期手动归档麻烦。

## 🚀 安装步骤

1. 在本机将本仓库的全部代码 Clone 或下载到任一目录（例如 `openflow/`）。
2. 打开 Google Chrome 浏览器，在地址栏输入 `chrome://extensions/` 进入扩展程序管理页面。
3. 打开页面右上角的 **开发者模式** (Developer mode) 开关。
4. 点击左上角的 **加载已解压的扩展程序** (Load unpacked) 按钮。
5. 在弹出的本地窗口中，选择刚刚下载/存放的 `openflow` 文件夹，点击“选择文件夹”。
6. 成功加载后，您就能在浏览器的工具栏中看到 OpenFlow 插件的图标（🧩拼图图标中的小齿轮），建议将其**固定**在工具栏方便随时点击使用。

## 💡 使用说明

1. 登录并打开需要抓取数据单据详情的网页系统界面。
2. 确保页面的信息已经完全加载完毕，左侧边栏选中的目标单据处于高亮状态。
3. 点击 Chrome 工具栏上的 **OpenFlow 插件图标**，会弹出提取器控制台。
4. 插件会自动开始检索提取；在预览窗口您可提前查阅抓到的游戏名称、素材分类及尺寸需求信息，以及预演建立的对应命名文件夹结构。
5. 点击 **下载 CSV** 或 **下载 JSON** 生成您所需的结果文件。

## 📁 目录结构

```text
openflow/
├── manifest.json       # 核心配置文件 (Manifest V3)
├── popup.html          # 点击插件图标弹出的用户界面
├── popup.css           # 弹出界面样式
├── popup.js            # 核心业务界面交互与 CSV/JSON 生成下载逻辑
├── content.js          # 在网页上下文执行抓取文字/DOM和数据提取的核心引擎代码
├── icons/              # 存放着所有尺寸的插件图标
├── test.HTML           # 本地用于调试和验证抓取规则的离线样例页面
└── lib/                # (历史构件)存放外部依赖。注：核心已变更为纯原生
```

## 🛠️ 后续二次开发

代码使用纯原生 HTML、JS 和 CSS 编写。若需增添新的自定字段映射、修改导出表格格式的排列顺序：
- 修改数据抓取规则，请调校 `content.js` 中正则表达式和 DOM Query 逻辑。
- 修改导出格式的组合、固定文本和字段提取的默认Fallback规则，请调校 `popup.js` 对应 `downloadCsvBtn` 和 `downloadJsonBtn` 中对应的行构建 (`rowData`) 数组。修改后，在扩展设置列表里点击刷新按钮（🔄）即可测试效果。
